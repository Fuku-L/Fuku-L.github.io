2. 新建 Spring Boot 项目

建立 maven 空项目，在其中创建一个 module 项目 springboot_jpa。

  1. 项目依赖
    项目依赖选择 JPA 和 Web。注意由于教程使用比较旧的内容，需要 JPA 版本为 <version>2.1.0.RELEASE</version>（需要注意的是，jpa 和 web 依赖版本需要选择一致，否则会出现依赖冲突。无需解决依赖冲突可以跳过 2.1 小节。）

项目可以不用使用 spring-boot-starter-parent 也不用 spring-boot-starter, 因为在引入 JPA 和 Web 的时候，也都有自动引入这两个。

现在，只需要 Spring Boot 的约定配置那样，进行包文件的创建和 xxApplication.class 的创建即可。配置步骤查看 2.2 小节内容，具体代码查看 项目 highlight_spring_jpa 下的子项目 springboot_jpa。

启动成功之后，可以使用子项目根目录下的 rest-api.http 文件测试功能。

2.2 项目配置步骤解析

  1. 配置基本属性
    在 application.properties 中配置数据源和 jpa 的相关属性
  2. 定义实体类映射
    在 domain 包下，编写数据库实体 Person.class，提供构造方法和 getter/setter 方法。
  3. 定义数据访问接口
    在 dao 包下，编写数据访问接口，interface，定义数据查询语句
  4. 编写 mvc 控制器
    DI 注入数据访问接口，调用相关接口获得数据，并返回成 json 到前端中。

具体代码查看 项目 highlight_spring_jpa 下的子项目 springboot_jpa的对应层内容。启动成功之后，可以使用springboot_jpa项目根目录下的 rest-api.http 文件测试对应接口的功能。
