Spring Boot

构建 SpringBoot

构建 SpringBoot 项目的方式有多种，通过官方网站 http://start.spring.io 生成项目包，通过 Spring Tool Suite，IDEA，Spring Boot CLI，Maven 手工构建。直接通过 spring.io 选择提前确定

本案例选择Maven 手工构建，步骤如下：

  1. 新建空maven
  2. 引入springboot-starter父依赖， 引入starter pom 的 web 依赖
  3. 建立包结构，建立 springboot 入口类（命名规则为：’artifactId + Application.class’）
  4. 测试项目部署是否正确：建立 controller 类进行测试。

具体实例代码查看 highlight_springboot 项目下，git log 为 ‘springboot项目基础构建’ 的提交内容。
