基于 Spring Boot 进行 web 开发

1. SpringBoot 的 Web开发支持

在 pom.xml 的依赖中，直接使用 starter-web 即可导入 spring-mvc 和 tomcat 依赖。

SpringBoot 的 autoconfiguration 配置的类会自动进行相关配置

2. Thymeleaf 模板引擎

Thymeleaf 是一个 Java 类库，一个 xml、xhtml、html5 的模板引擎。

  1. 引入 Thymeleaf

使用时候需要在 pom.xml 中导入starter-thymeleaf依赖， 标签需通过 <html xmlns:th="http://www.thymeleaf.org"> 引入命名空间。
基本的 Thymeleaf 模板页面如下，引入了 Bootstrap（css样式控制） 和 jQuery（DOM操作）【不需要可以去掉】

  1. 访问 model 中的数据
    通过 “${}” 访问 model 中的属性，需要处理动态内容需要加上”th:”前缀
  1. model 中数据的迭代
    通过 “th:each”来做循环迭代
  1. 数据判断
    使用 “th:if” 进行条件判断，支持 “>”,”<”,”>=”,”<=”,”==”,”!=”
    支持使用 SpringEL 表达式语言用于条件判断中。
  1. 在 JS 中访问 model 值
  • 通过 th:inline="javascript" 添加到script，在 JS 代码中可以访问 model 中的属性
  • 通过 “[[${}]]” 格式获得实际的值
