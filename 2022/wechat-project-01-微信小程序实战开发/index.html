小程序入门知识

1.小程序的概念和应用前景

不用安装，通过扫描二维码或搜索打开就可以使用的应用程序。

小程序特点：

  1. 不用安装
  2. 应用体积较小：上限是 2048KB ( 2MB )
  3. 跨平台

应用前景：

​ 适合用来开发一些业务逻辑简单、低频使用、对性能要求不高的应用程序。

发展特点：

1. 生态系统庞大
 2. 开放能力越来越强，能够匹配多种用户和场景。
 3. 用户黏性高

2.*开发环境的搭建和工具使用

小程序开发环境的搭建

  1. 下载安装“微信开发者工具”
  2. 在 微信公众平台 注册申请公众号、并开通开发者权限
  3. 打开开发者工具，使用微信登录后，单击“小程序项目”，就可以创建一个小程序项目
  4. 根据页面提示进行小程序项目的创建，选择项目目录，填写 AppID、项目名称，点击“确定”按钮即可。
  5. 创建成功后，会进入开发调试工具界面。

开发调试工具使用

微信开发者工具的界面主要分为 5 个部分，分别是：菜单栏、工具栏、模拟器、编辑器和调试器。

  1. 菜单栏：提供大部分工具功能入口
  2. 工具栏：提供一些常用功能的快捷按钮
  3. 模拟器：用来模拟手机运行环境，可以配置不同型号手机的运行效果，还可以切换网络环境
  4. 编辑器：类似于 vscode 软件的代码编写环境。
  5. 调试器：类似于 chrome 浏览器中的开发者工具

3.*小程序项目结构

小程序项目分为 4 个部分，包括：项目配置、应用程序配置、页面配置目录、公共脚本目录。

  1. 项目配置：一个 project.config.json 文件，用来管理项目的各种配置信息，包括项目名称、版本、依赖库等
  2. 应用程序配置：有三个文件：逻辑文件 app.js，配置文件 app.json，公共样式文件 app.wxss
  3. 页面配置目录：用来存放小程序的各个页面。每个页面是一个文件夹，文件夹中存放同命名的三个文件：逻辑、结构、配置。
  4. 公共脚本目录：存放公共方法的脚本，方便各个页面引入和使用。
